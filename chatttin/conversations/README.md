# Conversation Archive

**Save important conversations here for future reference.**

## How to Use

1. After a helpful conversation, copy it to a new file
2. Name it with date and topic: `YYYY-MM-DD-topic.md`
3. Add a brief summary at the top

## Example Format

```markdown
# 2026-01-19 - Setting Up Project

**Topic:** Initial project setup
**Model Used:** Auto (Claude Sonnet)
**Key Takeaway:** [What you learned]

## Conversation

[Paste conversation here]
```
